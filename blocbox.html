Project Overview

Blocbox is a responsive, software as a service (SaaS) web application built for collecting links, documents and images in a single location online, allowing users to collaborate with others online.
Go to the Website

The Challenge

As users access content through various devices today, they are storing and sharing more information online. Taming webs of information and organizing files often takes time away from real work creatives and business owners need to accomplish daily goals. Blocbox aims to resolve these issues by providing an accessible, easy-to-use solution, with a simple UI that allows the user to collect files and info in a single place.
